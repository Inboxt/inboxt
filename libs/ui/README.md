# Inboxt UI Component Library (@inboxt/ui)

This library contains shared React components and styling for Inboxt applications (Web app and Browser Extension).

## Tech Stack

- **Framework:** React
- **Styling:** Mantine (using shared themes and CSS Modules)

## Usage

Components are exported from the main `index.ts` and can be used in the `web` or `web-extension` apps.

## Shared Theme

The `src/theme` directory contains the base Mantine theme configuration, including custom colors, spacing, and font sizes used across the platform.
