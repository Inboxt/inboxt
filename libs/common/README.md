# Inboxt Shared Logic (@inboxt/common)

This library contains shared TypeScript types, validation schemas (Zod), and utility functions used by both the backend API and the frontend/extension.

## Usage

- **Backend:** Imported directly via TypeScript paths.
- **Frontend/Extension:** Used for type safety and consistent validation.

## Development

If you add new shared types or schemas, you may need to rebuild or restart the development servers to pick up the changes.
