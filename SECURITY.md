# Security Policy

We value the security of Inboxt and its users. If you discover a security vulnerability, we'd appreciate it if you could report it to us privately so we can address it as quickly as possible.

## Reporting a Vulnerability

Please report security issues directly to **security@inboxt.app**.

To help us understand and resolve the issue faster, please include:
- A brief description of the vulnerability.
- Steps to reproduce the issue (including any specific configuration).
- Potential impact and how it might be exploited.
- Any proof-of-concept or screenshots (if applicable).

**Important:** Please do not report security vulnerabilities via public GitHub issues.
