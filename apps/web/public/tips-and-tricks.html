Tips & Tricks

Small habits and shortcuts that make Inboxt easier to use over time.

Save faster

  • On desktop, the browser extension is the most reliable way to save pages.
  • On mobile, sharing to Inboxt is often faster than copying links.
  • If a page fails to parse, trying a different saving method can help.

Work in batches

  • Select multiple items to archive, label, or delete them at once.
  • Archiving regularly keeps your Inbox focused on unread content.

Use labels intentionally

  • Labels work best when they represent topics or long-term interests.
  • You can apply multiple labels to the same item.
  • Labels appear in navigation, making them useful as saved views.

Search like a power user

  • in:archive: search only archived items
  • label:Research: filter by label
  • type:newsletter or type:article
  • has:highlights: items with highlights
  • site:example.com: filter by source

Search filters can be combined to narrow things down quickly.

Highlights are lightweight

  • Highlights are not notes, they’re meant for quick reference.
  • Use exports if you want to move highlights into another tool.

Know the limits

  • Some dynamic or unusual pages may not extract perfectly.
  • Storage and feature limits exist to keep the service stable.

Full details are available in the documentation if you need them.
